
# Builders Connect

**Live, Earn, Grow Beyond Borders.**

Builders Connect empowers ambitious people to design lives beyond default settings—through practical strategies for mobility, wealth creation, and personal evolution.

## 🚀 Key Features

### 1. AI-Powered Newsletter Studio (Gemini 3 Pro)
- **Trend Discovery**: Uses Google Search Grounding to find real-time shifts in visa laws, tax policies, and global job markets.
- **Smart Composing**: Drafts high-converting, professional briefings for over 50,000 members based on current events.

### 2. Community Hub (Gemini 3 Flash)
- **AI Strategist**: Real-time multi-perspective opinion generator for relocation scenarios.
- **Persistent Threads**: A full-featured discussion engine with state persistence.

### 3. Decision Engineering Tools
- **Global Readiness Quiz**: A 3-minute assessment with radar chart visualization (Recharts).
- **Cost of Living Matcher**: Side-by-side data comparison of global cities.
- **Wealth Simulator**: Real-time tax impact and savings surplus calculator.

### 4. Enterprise CMS
- **Granular Role Management**: Admin, Back Office, and Member roles with specific Editor permissions.
- **Asset Control**: Full lifecycle management for guides, courses, and digital blueprints.

## 🛠 Tech Stack

- **Framework**: React 19 (ESM)
- **Styling**: Tailwind CSS
- **AI Engine**: Google Gemini API (`@google/genai`)
- **Intelligence**: Gemini 3 Pro (Complex Reasoning) & Gemini 3 Flash (High Latency)
- **Visualization**: Recharts
- **Icons**: Lucide React
- **Routing**: React Router 7 (Hash-based for easy deployment)

## 📦 Setup & Installation

1. **Clone the repo**:
   ```bash
   git clone https://github.com/your-username/builders-connect.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file in the root:
   ```env
   API_KEY=your_google_gemini_api_key
   ```

4. **Launch**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

The application is configured to run on GitHub Pages using `HashRouter`. No server-side configuration is required for routing.
